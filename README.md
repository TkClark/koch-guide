# Koch Chemie 考赫化学产品使用指南 (Products Guide)

**在线访问 (Online Preview): [https://tkclark.github.io/koch-guide/](https://tkclark.github.io/koch-guide/)**

这是一个为 **Koch Chemie (考赫化学)** 全体系产品专门设计的交互式网页端产品使用说明书。该页面收录了 134 款考赫化学的专业汽车美容护理产品，通过直观的颜色分类、便捷的检索功能以及详尽的产品详情展示，帮助汽车美容专业人士和爱好者快速检索并了解每款产品的使用方法、稀释比例及注意事项。

## ✨ 主要特性 (Features)

- 🎨 **官方颜色分类系统**：完美还原 Koch Chemie 官方的 10 大颜色分类体系，一目了然。
- 🔍 **实时检索功能**：支持通过产品名称、缩写代号 (如：F6.01, Gs 等) 或中文用途进行实时模糊搜索。
- 📱 **响应式设计 & PWA 支持**：适配桌面端与移动端，并支持作为 PWA 应用程序添加至 iOS/Android 主屏幕，实现原生 App 般的离线式浏览体验。
- 🏷️ **交互式图例过滤**：点击顶部的颜色图例，即可快速筛选查看指定类别下的所有产品。
- 📖 **详尽的产品档案 (弹窗展示)**：
  - 产品中英文名称及代号
  - 产品功能简介
  - 动态显示产品的 pH 值属性 (酸性 / 中性 / 碱性)
  - 核心配比数据：PA 壶比例、喷壶比例、原液使用建议
  - **官方建议与个人经验分享**：结合实战经验，给出最接地气的施工指导
  - 具有针对性的关键注意事项警告

## 🌈 类别说明 (Product Categories)

考赫化学的产品线通过瓶身上的标签颜色进行了科学的系统化分类：

1. 🟢 **Green (绿标)**：碱性清洁剂 / Alkaline cleaners (核心碱性预洗与全能清洁)
2. 🟡 **Yellow (黄标)**：洗车香波 / Shampoos (温和与洗车香波)
3. 🔴 **Red (红标)**：酸性及轮毂清洁 / Acid cleaners & rim cleaners (强效轮毂与漆面酸洗去氧化)
4. 🟠 **Orange (橙标)**：溶剂与除胶 / Solvents (柏油、虫尸、油污等强效溶解)
5. 🔵 **Blue (蓝标)**：玻璃清洁与水处理 / Glass cleaners & water processing (高效玻璃清洁及系统洗车水处理)
6. 🩵 **Cyan (青标)**：内饰塑料与皮革 / Leather & plastics interior (汽车内部织物、皮革与塑料养护)
7. 🔘 **Gray (灰标)**：外饰橡胶与塑料 / Rubber & plastics external (车辆外部塑料、橡胶和发动机舱涂层养护)
8. ⚫ **Black (黑标)**：抛光与漆面护理 / Body & paint (漆面研磨剂、抛光垫与镜面封体)
9. 🟣 **Purple (紫标)**：香氛系列 / Fragrances (车内异味消除与高级香氛)
10. 🫒 **Olive (橄榄绿标)**：驱水与保固剂 / Driers & conservers (洗车房水蜡、漆面驱水与快速镀膜增艳)

## 🛠️ 技术栈 (Tech Stack)

本项目采用纯前端技术栈构建，轻量且无需后端支持：
- **HTML5**: 语义化结构，并包含 iOS PWA 适配的 Meta 标签。
- **CSS3**: 使用 CSS Variables 管理主题色，Flexbox & Grid 进行现代化响应式布局，毛玻璃及平滑动画增强 UI 质感。
- **Vanilla JavaScript**: 处理全部的 DOM 交互、实时搜索过滤、分类筛选及模态框动态数据渲染。产品数据以 JS Object 数组的形式内嵌，便于离线访问和维护。

## 🚀 如何使用 (Usage)

1. **直接打开**：双击使用任何现代浏览器（Chrome, Safari, Edge 等）打开 `KochChemie_Products_Guide.html` 文件即可直接使用。
2. **手机端体验最佳化 (iOS)**：
   - 将该 HTML 文件部署至任意 Web 服务器。
   - 使用 iPhone 的 Safari 浏览器打开链接。
   - 点击浏览器底部的 "分享" 按钮，选择 **"添加到主屏幕" (Add to Home Screen)**。
   - 即可在桌面生成一个类似原生 App 的图标，随时享受全屏阅览体验。
